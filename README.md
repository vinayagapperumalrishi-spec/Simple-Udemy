# 📘 Udemy Clone (Basic Frontend)

This is a simple **Udemy-inspired website** built using **HTML and basic CSS**. The project focuses on practicing frontend fundamentals by recreating a clean and structured layout similar to an online learning platform.

## 🚀 Features

* 📄 Two static pages:

  * **Home Page** – Displays courses and general layout
  * **Course Page** – Shows course details
* 🎨 Styled using basic CSS (no frameworks)
* 📱 Simple and responsive layout design
* 🧩 Beginner-friendly structure

## 🛠️ Technologies Used

* HTML5
* CSS3 (Basic styling)



## 🎯 Purpose

This project was created to:

* Practice HTML structure and layout
* Understand CSS styling and positioning
* Build a basic multi-page website

## 📌 Future Improvements

* Improve responsiveness for mobile devices
* Add more pages (Login, Signup, Dashboard)
* Enhance UI design
